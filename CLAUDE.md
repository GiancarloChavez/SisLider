## Project Overview

SisLider is an academic Systems Architecture (ASI) project documenting and implementing an educational institution management system using the TOGAF framework (Group 2).

## Architecture Domains

The system is organized around four main domains, each with ASIS/TOBE diagrams:

- **Matricula** — Student enrollment management
- **Asistencias** — Class attendance tracking
- **Clases** — Course and class management
- **Pagos** — Payment processing

## Project Status

Early stage — architecture diagrams exist (Draw.io) but implementation has not started. Technology stack and build tooling are yet to be defined. Update this file once the stack is chosen and the project is initialized.
